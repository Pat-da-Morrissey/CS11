This code is a simpler replica of the 1970's game called Oregon Trail. 
This texted based game makes you go down the Oregon Trial and try to survive no matter how hard the condtions might be.

To play the game just follow the prompts given below and your input will determine your survivability

To compile your program you must complie the files main.cpp as well as the Player.cpp to run the game you can run the Game.exe file
